About This Project

Amazon Web Services (AWS) is a cloud computing platform that provides scalable, reliable, and cost-effective services for hosting applications and storing data.

Services Used

Amazon S3

Amazon S3 (Simple Storage Service) is used to store and host the static website files such as HTML and CSS. It provides high availability and low cost hosting.

AWS IAM

AWS Identity and Access Management (IAM) is used to securely control access to AWS resources. A user with minimal permissions was created to manage the S3 bucket.

Static Website Hosting

S3 static hosting allows the website to be publicly accessible through a URL without needing a backend server.
